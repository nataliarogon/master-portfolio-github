BAUHAUS STYLE THEMED WEBSITE

"Good architecture should be a projection of life itself, and that implies an intimate knowledge of biological, social, technical, and artistic problems." ~ Walter Gropius

The website was created for the purpose of KEA project for the very first web design week. I never got a chance to code anything before, so I was facing a lot of difficulties while creating htis assigment. The page was created in a spirit of bauhaus- major architectural style strongly related to Modernism and Modern architecture, created by Walter Gorpius.

Design wise it's red, yellow and blue which are the main color shames used for the website. Beacuese of divercity of those colors, backround and font are presented in black and white contrast so the text is easily readible and does not seem overwhelming for the receiver.
